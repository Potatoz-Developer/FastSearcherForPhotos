FastSearcherForPhotos is a fast and simple web tool to find images and GIFs instantly based on theme, style, and source.

## Features

-Instant search generation
-Theme + style combination

## --Multiple sources:--

-Google Images
-Pinterest
-Tenor (GIFs)
## -------------Fast Acess-----------------------------
-One-click search
-Install shortcut for faster access


##  How to use

1. Select a **theme**
2. Choose a **style**
3. Pick a **source**
4. Click **Search now**
